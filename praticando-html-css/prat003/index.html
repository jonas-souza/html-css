Como adicionar Favicon no seu site.

Como adicionar

Para adicionar um Favicon no seu site, você precisa digitar a tag <link> (abaixo da tag <title>) e selecionar a opção favicon.

Logo em seguida, você selecione a imagem salva nos arquivos dentro das aspas e digite a referência da imagem.

Qualquer imagem serve para usar como Favicon?

Não!
A imagem deve ser sem muitos detalhes e no formato .ico.
