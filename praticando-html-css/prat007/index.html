    Marcando um trecho do texto

    Para marcar um trecho importante de um texto, usaremos a tag <mark></mark>:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Eum voluptate eaque consequatur! Molestias distinctio at tempore nam? Sequi, reiciendis autem qui doloribus incidunt, dicta obcaecati dolor amet magni eum a?

    Para mudar a cor do marcador, digitamos o seguinte código:

    <mark style="background-color:(a cor que deseja alterar);">

    Texto deletado e inserido

    Para acrescentar um trecho do texto deletado, utilizaremos a tag <del>.
    Ex.:

    Você pode ler tudo aqui, menos isso aqui, por que é IRRELEVANTE!

    Para digitar um texto inserido, utilizaremos a tag <ins>.
    Ex.:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Sed suscipit quasi laborum, inventore eveniet rerum veniam itaque accusantium quae velit eos, soluta aliquam corporis est architecto vel laudantium labore dolores?

    Texto Grande e Pequeno

    Eu quero que um trecho do parágrafo fique com a fonte grande, usa-se a tag (já obsoleta) <big>, porém, devemos utilizar a CSS para modificar, já que HTML5 trabalha com semântica. Ex.:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Dignissimos mollitia ipsum inventore amet voluptate dolor possimus consequatur consequuntur ea magnam et, ducimus repellendus pariatur quisquam modi iste id tempore repellat?

    Agora quero que um trecho do parágrafo tenha a letra(fonte) pequena (small).
    Ex.:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Reprehenderit similique magnam aliquam, aut ipsum consectetur saepe repellendus earum, magni blanditiis atque illum expedita delectus nostrum tenetur maxime necessitatibus? Optio, unde!
